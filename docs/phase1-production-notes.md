# Phase 1 production task subset

This list is derived from the Phase 1 core set after a targeted weak-task rerun.

Excluded from the first production corpus:
- identify-shape
- click-dialog-2
- navigate-tree

Reason:
- these remained materially weaker than the rest of the local-Qwen task set in targeted evaluation
- excluding them should improve signal quality for the first SFT corpus
- they can be collected later in a separate lower-confidence bucket if we want broader coverage
